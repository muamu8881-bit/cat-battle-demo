# 音效资源（可选）

游戏会尝试从本目录加载 **SFX** 用 MP3；**缺文件**时用 **Web Audio 程序合成** 发声。  

**背景音乐（可选）**：在下列路径放入自己的 **`bgm.mp3`** 即可循环播放；没有该文件时**无声**（无程序垫音）。首次需用户点击/关闭引导后才会因浏览器自动播放策略开始播放。

## 文件名与用途

| 文件 | 用途 |
|------|------|
| `bgm.mp3` | 循环背景音乐（`volume` 在代码中约 0.24，可自行改 `index.html` 里 `bgm.volume`） |
| `hit.mp3` | 子弹/激光命中怪物 |
| `fire.mp3` | 发射主弹 |
| `explode.mp3` | 炸弹/火球爆炸范围伤害 |
| `hurt.mp3` | 玩家受击（血条被敌弹命中、怪物压线） |
| `win.mp3` | 通关胜利 |
| `lose.mp3` | 失败 |
| `pick.mp3` | 肉鸽选卡确认 |
| `chest.mp3` | 肉鸽宝箱/翻牌界面打开 |
| `laser.mp3` | 狸花横向激光扫过（与 `hit` 可区分设计） |
| `ui.mp3` | 预留（当前逻辑未强制使用，可今后接 UI 点击等） |

版本号：在 `index.html` / `chinaindex/index.html` 中调整 `SOUND_ASSET_VER`，与 `?v=` 查询参数一起用于缓存刷新。

## 部署说明

- 与图片相同，将本目录 **完整上传** 到站点 `assets/sounds/` 下，保证 URL 与 `assetUrl('assets/sounds/...')` 一致。
- 若 `ASSET_BASE` 指向 COS 等跨域域名，页面已为 `Audio` 设置 `crossOrigin = "anonymous"`；需确保存储桶 **CORS** 允许你的页面来源。

## 素材来源提示

可选用 CC0 / 明确可商用的音效包，自行重命名覆盖上表文件名即可。
